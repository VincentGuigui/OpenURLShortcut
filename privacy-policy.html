Privacy Policy

Open URL Shortcut — Last updated: April 2026

Overview

Open URL Shortcut is a client-side web application. It does not collect, transmit, or store any personal data on any server.

Data access

This app requests access to Google Drive using the drive.file scope, which grants access only to files the user explicitly opens with this app. The app cannot read, list, or access any other files in your Google Drive.

How file data is used

When you open a .url file with this app, its contents are read directly in your browser to extract the URL it contains. The file content is never sent to any server and is discarded as soon as the page is closed or you navigate away.

Authentication tokens

The OAuth access token obtained from Google is stored only in browser memory (a JavaScript variable). It is never written to localStorage, cookies, or any external service. It expires automatically after one hour.

Third-party services

This app loads the Google Identity Services library from accounts.google.com and calls the Google Drive REST API at googleapis.com. These services are subject to Google's Privacy Policy.

No analytics or tracking

This app contains no analytics, tracking pixels, advertising scripts, or any form of usage monitoring.

Contact

If you have questions about this privacy policy, please open an issue in the project repository.

This policy applies to the Open URL Shortcut app published on Google Workspace Marketplace.
